금요일 업로드 체크리스트

FTP 업로드와 함께 `load level`을 전환하면 `/api/load/:level`이 호출되고 `/api/metrics`가 갱신됩니다. 업로드 트래픽·파일을 기록할 때 활용하세요.

  • Warm-up: 베이스라인 RPS/지연 확보
  • Normal: 평균 지연과 CPU 안정성 확인
  • Peak: 업로드 폭주 시 경고 배너 확인
  • Overload: 의도적 포화(오류율/지연 급증)

모니터링 포인트 (FTP/HTTP)
