PeakLoadVisualizer · Docs

폴더 및 파일 구조

리포지토리: GitHub

루트

  • README.md — 프로젝트 개요와 실행 방법
  • package.json, tsconfig*.json — 빌드/의존성/TS 설정
  • docs/ — 개요, 윈도우 셋업, 부하 계획 문서
  • src/ — TypeScript 소스 (서버/클라이언트)
  • public/ — 빌드된 정적 자산 배포 디렉터리
  • dist/ — 서버 빌드 산출물

서버 (src/server)

  • app.ts — Express 서버, 정적 서빙, 메트릭/로드/리더보드/닉네임 API
  • metrics.ts — 시스템 실측(CPU/메모리)과 요청 지연/오류율을 집계하여 `/api/metrics` 응답 생성

클라이언트 (src/client)

  • index.html — 대시보드 UI, 부하 단계 전환, 리더보드 테이블, i18n, 도움말 툴팁
  • styles.css — 레이아웃/테마/차트/툴팁 스타일
  • main.ts — 메트릭 폴링, 부하 전환, 닉네임 저장, 리더보드 렌더링, 언어 전환, 도움말 로직
  • docs.html — 현재 페이지(파일 구조 요약)

문서 (docs/)

  • overview.md — 시나리오/기능/스택 개요
  • windows-setup.md — VMware + Windows Server 2016 세팅
  • load-plan.md — 부하 테스트 계획 및 메트릭 관측

빌드/실행

npm install
npm run build
npm start

개발 모드: npm run dev (ts-node + 클라이언트 watch)
